변수

변수란 컴퓨터의 기억공간에 편하게 붙인 이름이다

컴퓨터의 기억장치는 0이나 1로만 채워진다. 2진수 체계 메모리들은 0부터 차례차례 번호가 부여되고 이 번호를 주소라 한다
0번 주소에 값 100을 넣고 1번 주소에 값 200을 넣어라라고 하면 값을 넣은 사람도 본인이 어디에 값을 저장해두었는지 알지 못한다.
그래서, 대신에 기억하기 쉬운 이름을 붙이고 이걸 변수라고 한다.
변수명은 알파벳 대문자, _(밑줄문자), 숫자로 구성된다.
*단, 시작은 반드시 영대소문자나 _(밑줄문자)만 가능하다.
원래 변수는 시작전에 미리 다 만들어 놓고 써야 하나 자바스크립트는 필요할 때마다 만들어 쓰면 된다.
html은 대소문자 구분을 하지 않지만 자바스크립트는 대소문자를 구분한다

x1, X1는 서로 다른 변수이다
abs1, kor_score, kor1, eng_1 가능
1_eng 불가능
